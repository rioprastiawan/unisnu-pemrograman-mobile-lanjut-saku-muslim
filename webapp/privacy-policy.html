Kebijakan Privasi

Terakhir diperbarui: 26 Desember 2025

üîê Privasi Anda, Prioritas Kami

Saku Muslim adalah aplikasi 100% offline yang tidak mengumpulkan, menyimpan, atau membagikan data pribadi Anda ke server kami atau pihak ketiga. Semua data disimpan secara lokal di perangkat Anda.

1Data yang Kami Kumpulkan

üìç Data Lokasi

Aplikasi meminta akses lokasi perangkat Anda untuk:

  • Menentukan waktu sholat yang akurat sesuai koordinat geografis Anda
  • Menampilkan arah kiblat pada fitur kompas digital

Penting: Data lokasi hanya disimpan di perangkat Anda dan tidak pernah dikirim ke server manapun.

‚öôÔ∏è Data Preferensi & Pengaturan Lokal

Aplikasi menyimpan pengaturan dan preferensi Anda secara lokal, termasuk:

  • Pengaturan notifikasi adzan (waktu dan pilihan suara)
  • Bookmark surah dan ayat Al-Qur'an favorit Anda
  • Riwayat bacaan doa dan dzikir
  • Preferensi tampilan aplikasi

Teknologi: Menggunakan SharedPreferences dan SQLite database lokal yang tersimpan hanya di perangkat Anda.

2Bagaimana Kami Menggunakan Data

Data yang dikumpulkan hanya digunakan untuk meningkatkan pengalaman ibadah Anda:

  • Menampilkan waktu sholat yang akurat berdasarkan lokasi geografis Anda
  • Mengirim notifikasi pengingat adzan sesuai pengaturan yang Anda pilih
  • Menampilkan arah kiblat dengan akurat menggunakan sensor kompas
  • Menyimpan bookmark dan riwayat bacaan untuk kemudahan akses

Jaminan Kami: Kami tidak pernah menganalisis, membagikan, atau menjual data Anda kepada pihak manapun.

3Penyimpanan & Keamanan Data

Komitmen kami terhadap privasi Anda:

  • Tidak ada server backend - Kami tidak memiliki server yang menyimpan data pengguna
  • Tidak ada sinkronisasi cloud - Data tidak di-backup atau di-upload ke cloud manapun
  • Data tetap di perangkat Anda - Hanya Anda yang memiliki akses penuh ke data Anda
  • Penghapusan otomatis - Jika Anda menghapus aplikasi, semua data akan terhapus dari perangkat

4Izin yang Diperlukan

Untuk memberikan layanan terbaik, aplikasi memerlukan izin berikut:

  • üìç Akses Lokasi - Menentukan waktu sholat akurat dan arah kiblat
  • üîî Notifikasi - Mengirim pengingat waktu sholat dan adzan
  • üåê Internet - Mengunduh data Al-Qur'an dan audio murattal (hanya sekali saat pertama)
  • üß≠ Sensor Kompas - Menampilkan arah kiblat dengan akurat

Semua izin dapat dikelola melalui pengaturan perangkat Anda kapan saja.

5Layanan Pihak Ketiga

Aplikasi menggunakan API eksternal berikut untuk menyediakan konten:

  • equran.id - Menyediakan teks Al-Qur'an lengkap dengan terjemahan Bahasa Indonesia
  • api.myquran.com - Menyediakan data jadwal waktu sholat berdasarkan lokasi

Catatan Penting: API ini hanya digunakan untuk mengunduh konten dan tidak mengumpulkan atau menyimpan data pribadi Anda.

6Hak & Kontrol Anda

Anda memiliki kontrol penuh atas data Anda:

  • Hapus semua data - Cukup hapus aplikasi untuk menghapus seluruh data lokal
  • Kontrol izin - Nonaktifkan izin lokasi atau notifikasi kapan saja melalui pengaturan
  • Mode offline penuh - Gunakan aplikasi sepenuhnya offline setelah data Al-Qur'an terunduh
  • Transparansi - Tidak ada data tersembunyi atau tracking yang berjalan di background
